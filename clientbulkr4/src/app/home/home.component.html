FHIR Bulk Data API Example Code

This sandbox offers example code and demonstration capabilities following the emerging FHIR bulk data implementation guide. The current version of the developing specification is available at https://github.com/smart-on-fhir/fhir-bulk-data-docs

Beginners Looking to Learn

Try the Bulk Data API's Simulation and inspect the API request and response for each step in the sequence. Note how the body of the response points to the ndjson (newline delimited JSON) resources. Download and inspect the sample data returned.

Experts Looking to Test

The sandbox contains three components as described below:

  1. A SITE FHIR Server that conforms to the Bulk Data API Server implementation. The code base can be found at
    • https://github.com/onc-healthit/fhir-tools/tree/master/bulkr4
  2. A FHIR Client that conforms to the Bulk Data API Client Implementation.The code base can be found at
    • https://github.com/onc-healthit/fhir-tools/tree/master/secure-backend-app
  3. This User Interface (not part of the specification.) If you are new to the Bulk Data API's, see the “Bulk Data API's Simulation” on the left navigation menu.The code base for the User Interface can be found at
    • https://github.com/onc-healthit/fhir-tools/tree/master/clientbulkr4

The User Interface interacts with Bulk Data Client using RESTful interfaces, which in turn communicates with the SITE FHIR Server using the FHIR Bulk Data APIs to satisfy the requests.

The SMART Backend Services authorization guide is implemented between the Bulk Data Client and the SITE FHIR Server.
